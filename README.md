# Gemini CLI Env Setup

モビリティの高い「Gemini CLI ＋ VSCode」のローカル開発環境を、素早く確実にセットアップするためのリポジトリです。
手動作業とAI（Gemini）によるナビゲートを組み合わせることで、環境差異を吸収し、どのPCでも素早く同じ開発体験を得ることを目指します。

## 🎯 ゴール（目指す状態）

以下のツール群がインストール・認証され、正常に連携できる状態（オールグリーン）を目標とします。

1. Git Bash が使える（Windowsの場合）
2. Node.js が使える
3. gh コマンドが使える（認証済）
4. git コマンドが使える（gh認証と連携済）
5. Dockerデーモンが起動し、応答する
6. gcloud CLI が使える（ADC認証済・プロジェクト設定済）
7. Terraform コマンドが使える
8. VSCode が使える（インストール済）
9. VSCode の設定・拡張機能が同期・反映されている
10. Gemini CLI が使える（認証済）
11. Gemini MCP（Google Developer・Terraform等）が稼働する

## 🚀 セットアップの進め方

具体的なセットアップ手順については、[SETUP.md](./SETUP.md) を参照してください。
まずはWeb版のGeminiに `SETUP.md` を読み込ませて、ナビゲートを受けるところからスタートします。
